CSS Selectors

Universal selector

selects all the elements of the DOM, denoted by asteric i.e. " * "

Type selector

selects specific tags of an HTML page, like - h1, span, div, p, button etc.

Class selector

selects one or more elements that has an unique class attribute, one or more elements may have a same class name, denoted by a dot i.e. " . "

ID selector

selects specific element that has an unique ID name, an element should have only one unique ID in that HTML page, denoted by " # "

Attribute selector

selects those elements that have selected attributes and their values, ex. - input[type="text"], this selects only those input tags that have a type attribute with "text" value

Descendant selector

selects those elements that are descendants of the previous root element, ex. - article div p , selects those p tags that are descendants of an article tag

Child selector

selects that element which is an exact child of its previous element, ex. - div > p, selects those p elements that are exactly the children of div elements

Adjacent Sibling Selector

selects the adjacent elements of the same parent element, ex. - h1 + p, selects those h1 and p elements that are adjacent siblings of the same parent element

General Sibling Selector

selects those sibling elements that may or may not be adjacent but belongs to the same parent element, ex. - h2 ~ p, selects h2 and p adj or non-adj elements

Pseudo-class Selector

used to apply special actions on an element, ex. - button : hover, used to apply hover action on button element

Pseudo-element Selector

used to apply special actions on an element, ex. - p :: first-letter, to work on the first letter of a paragraph element

Group selectors

selects a group of elements to aaply styles on them, ex. - h1, h2, h3 {color: red}, apply red text colors on all h1, h2, and h3 elements
